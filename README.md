Verinode
Verinode is an open-source Web3 infrastructure for issuing and verifying cryptographic proofs of real-world events. It provides secure, decentralized, and tamper-proof verification using smart contracts and blockchain technology.

Features
- Issuing cryptographic proofs for events
- Verifying proofs on-chain
- Wallet integration (MetaMask, WalletConnect)
- Modular backend and frontend architecture
- Open-source and community-driven
